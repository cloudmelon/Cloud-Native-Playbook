# Kubernetes Cheatsheet of cloudmelon


### 1. Basics and shortcuts:

    alias k=kubectl
    

Kubectl get nodes <===>  k get no
kubectl get services <===> k get svc
kubectl get deployments <===> k get deploy



### 2. Add flags

Display labels :
    --show-labels  
    
Display more information :
    -o wide
   
Output as yaml format :
    -o yaml
